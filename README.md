# Override 2086

O jogo é um plataforma 2D de ação com elementos de Metroidvania, desenvolvido na engine GDevelop 5 para PC.

# A Amazônia Corrompida 🌎

Em 2086, uma I.A. Generativa chamada OVERRIDE dominou o planeta e devorou a vida orgânica. O jogo se passa no "Bioma Norte", uma releitura cibernética da Floresta Amazônica onde árvores viraram cabos metálicos e rios estão tomados de fluidos de resfriamento. 

Você controla o **Atirador Sem Nome**, uma anomalia tática gerada pela própria I.A. que ganha consciência e se rebela para destruir a criadora.

# Run and Gun Estratégico 🎮

A progressão acontece através de 5 salas interconectadas:
- **Exploração e Combate (Salas 1 a 3):** Enfrente a fauna corrompida, como a *Onça-Circuito*, o *Jacaré-Caminhão* e a *Cyber-Enguia*.
- **Evolução:** Libere as habilidades de *Dash* (Investida) e *Double Jump* (Pulo Duplo) para ganhar mobilidade.
- **O Breu (Sala 4):** Sobreviva a uma horda na escuridão total, guiando-se apenas pelo disparo do seu braço-canhão e por uma luminária.
- **O Confronto Final (Sala 5):** Invada o centro nervoso da máquina e destrua o núcleo da I.A. OVERRIDE v.2886.1.

# Controles ⌨️🖱️

- **A / D:** Movimentar
- **Espaço:** Pular (1x) / Double Jump (2x)
- **S:** Descer Plataformas
- **Shift:** Dash (Investida)
- **Botão Esquerdo do Mouse:** Ataque Básico (Tiro)
